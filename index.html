For my project proposal, I would like to make a neural network that can predict the outcome of a tennis match between two professional tennis players. This model would be trained on previous match data to be able to predict who wins certain matches. The part of this project that will be difficult will be finding enough data that is compatible with the model. Goals: 1. Have the model be at least 70% accurate. 2. Find data that goes back to at least 2005. 3. Have the model work based on different conditions (court surface, country match is played in, etc.)
